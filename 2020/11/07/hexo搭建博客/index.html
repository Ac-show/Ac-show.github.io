GitHub + Hexo 搭建博客

这是我一边搭建一边写的，按照步骤来应该没什么问题

1.准备环境

安装git

去官网下载安装就行

安装node

去官网下载安装

安装hexo-cli

2.本地安装Hexo

2.1搭建hexo

找一个目录新建一个文件夹用于存放你的blog项目

打开控制台，cd到你的文件夹下

2.2初始化项目

初始化项目后通过一下命令就可以在本地访问了

3.创建GitHub仓库

4.将GitHub仓库与hexo项目连接起来

4.1 生成ssh密钥

打开控制台输入

生成ssh成功后

4.2 设置GitHub

打开GitHub个人设置

测试连接GitHub

4.3 配置本地hexo项目下的_config.yml

4.4 博客发布到GitHub仓库

安装依赖

上传GitHub

现在可以直接访问了：GitHub用户名.github.io

注意：可能有缓存，要等一会

5.配置域名

先买一个域名，要不然还是用GitHub分配给你的域名把

先ping一下GitHub给你的域名

复制ip

解析域名

添加记录

创建CNAME文件

进入你博客目录下的source文件夹，新建一个CNAME文件，将你的域名写进去，CNAME没有后缀名！！

重新上传 hexo d

现在用你自己的域名应该是上面这张图片的样子

进入你GitHub仓库的setting下设置一下就好了

现在应该可以直接访问了
